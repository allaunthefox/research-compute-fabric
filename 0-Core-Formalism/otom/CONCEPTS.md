Ordered Transformation & Orchestration Model. The umbrella project label for ordered structure, transformation, orchestration, and model routing.

Endless Node Edges. Distributed substrate, publishing, credential/provenance, and mesh-routing family.

Frustration Aligned Memory Management. Route-memory layer that stores failed, partial, and successful paths. Principle: routing pain becomes routing memory.

Perfectly Imperfect Square Theory. Parallel non-orthogonal state exploration and imperfect-square/torsion routing frame.
