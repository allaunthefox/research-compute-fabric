# Filtered Resonance Remainder Averaging

## Purpose

This note extends the Hairy Ball FFT Topological Filter into a finite measurement procedure:

```text
filtered spectral product
-> residual/remainder field
-> scan candidate number sets against the remainder
-> score resonance
-> average over finite windows
-> produce candidate active index set
-> audit arithmetic uniqueness separately
```

The goal is to identify which finite number sets resonate with the filtered remainder of the signal after known modes, voids, defects, and noise bands have been removed.

This is a candidate-discovery layer, not a Sidon proof by itself.

## Core Statement

After applying FFT/GFT/Hodge/Dirac filters, the remainder is the part of the signal not explained by the known material modes or known void patterns.

```text
remainder = observed signal - filtered known structure
```

A number set is useful when its spectral fingerprint explains the remainder with high coherence and low residual error.

## Finite Signal Setup

For a finite sampled signal:

```text
f_N[n], 0 <= n < N
```

Compute the spectral transform:

```text
F_N[k] = FFT(f_N[n])
```

Apply a filter mask:

```text
F_filtered[k] = H_N[k] F_N[k]
```

where `H_N` can remove known noise, known material resonances, saturated modes, or irrelevant bands.

The remainder is:

```text
R_N[k] = F_N[k] - F_filtered[k]
```

or in the physical-domain reconstruction:

```text
r_N[n] = IFFT(R_N[k])
```

## Candidate Number-Set Fingerprint

For a finite candidate set:

```text
A = {a_1, ..., a_m}
```

construct its spectral fingerprint:

```text
S_A[k] = sum_{a in A} w_a exp(i * 2*pi*k*a/N)
```

Power fingerprint:

```text
P_A[k] = |S_A[k]|^2
```

Difference-spectrum expansion:

```text
P_A[k] = sum_a w_a^2 + 2 sum_{a<b} w_a w_b cos(2*pi*k*(a-b)/N)
```

Important note:

```text
P_A exposes A-A directly. Sidon/B2 still requires A+A audit.
```

## Resonance Score

Compare the candidate fingerprint to the remainder:

```text
Res(A;R_N) = <normalize(P_A), normalize(|R_N|^2)>
```

Expanded normalized correlation:

```text
Res(A;R_N) =
  sum_k (P_A[k]-mean(P_A)) (|R_N[k]|^2-mean(|R_N|^2))
  / (||P_A-mean(P_A)|| * |||R_N|^2-mean(|R_N|^2)||)
```

A high score means the set's induced difference spectrum explains the remaining energy.

## Averaging Rule

Because the signal is finite, noisy, and window-dependent, use a finite average over windows:

```text
AvgRes(A) = (1/J) * sum_{j=1}^{J} Res(A; R_{N,j})
```

where each `R_{N,j}` is a finite windowed remainder.

Robust version:

```text
MedRes(A) = median_j Res(A; R_{N,j})
```

Stability receipt:

```text
VarRes(A) = variance_j Res(A; R_{N,j})
```

Candidate acceptance:

```text
A is accepted iff
  AvgRes(A) >= theta_res(N)
  and VarRes(A) <= theta_var(N)
  and VoidFit(A) >= theta_void(N)
  and Repair(A) >= theta_repair(N)
```

## Remainder Resonance Gate

The active candidate set is generated by:

```text
I_candidate(N) = argmax_A AvgRes(A)
```

under constraints:

```text
A subset {1,...,N}
|A| approximately sqrt(N)
finite timelike window only
finite FFT samples only
finite material/recoverability receipts only
```

## Arithmetic Audit After Resonance

The resonance score does not prove Sidon. After selecting candidate `A`, run:

```text
DifferenceSetReceipt:
  |{ |a_i-a_j| : i<j }| = m(m-1)/2

SumSetReceipt:
  |{ a_i+a_j : i<=j }| = m(m+1)/2
```

If ordinary pair-sum uniqueness fails, a virtual nonseparable encoding may still be tested:

```text
S_N(a,b) = Phi_N(a) + Phi_N(b) + Lambda_N(a,b)
S_N(a,b)=S_N(c,d) -> {a,b}={c,d}
```

## Interpretation

This gives a precise meaning to:

```text
find where a number set resonates with the remainder and then find an average
```

The remainder is the unexplained spectral residue. A candidate number set is a finite arithmetic lens. The average is a stability measure over finite windows, not an infinite-data operation.

## Pipeline

```text
finite signal f_N
-> FFT/GFT/Hodge filter
-> remainder R_N
-> candidate number-set fingerprint S_A or P_A
-> resonance score Res(A;R_N)
-> finite-window average AvgRes(A)
-> void/topological defect consistency check
-> difference-set and sum-set audits
-> nonseparable encoding if needed
-> compact-density receipt
```

## Boundary

Do not claim:

```text
resonance proves Sidon
FFT remainder proves B2 uniqueness
average over infinite data
spectral correlation is empirical proof of density
```

Allowed claim:

```text
finite-window resonance against the filtered remainder can identify candidate number sets whose spectral fingerprints explain the residual signal; those candidates must then pass arithmetic and encoding audits.
```
