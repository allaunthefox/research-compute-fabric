`GLOSSARY.md`: each entry gives the evocative name, the neutral technical
meaning, the stack role, and the best current evidence surface.

The rule is simple: weird names are allowed when they help retrieval, but every

## Reading Rule

If a term sounds like mythology, poetry, or hardware folklore, translate it
before promotion:

1. Name the neutral mechanism.
2. Name the invariant or gate.
3. Name the receipt surface.
4. Mark unknown expansions as unresolved instead of filling them with vibes.
