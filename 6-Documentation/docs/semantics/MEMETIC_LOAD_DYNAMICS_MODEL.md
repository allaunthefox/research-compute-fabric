# Event Model: Memetic Load Dynamics (Epidemiology + Cognitive Load)

This model integrates the **Biological Spread Theory** with the **6-Component Cognitive Load Function** to define the propagation and containment dynamics of the Cognitive Landmine.

## 1. The Infectivity Coefficient ($\beta$): The "Legendary Lure"
Infectivity is the probability that the pathogen (the meme) captures an observer's attention and prompts a decoding effort.
- **$\beta \propto L_{intrinsic} + L_{routing}$**
- **$L_{intrinsic}$ (Complexity)**: High mathematical difficulty in the AngrySphinx protocol creates the "Cipher Lure." Observers assume high intrinsic load implies high germane value ($L_G$).
- **$L_{routing}$ (Accessibility)**: The ease of shard discovery.

## 2. The Incubation Period ($T_{inc}$): The Sunk Cost Period
The time an observer spends "decoding" before the "Radioactive" payload is reached.
- **$T_{inc} \propto \int (L_{intrinsic} + L_{memory}) \, dt$**
- **$L_{memory}$ (Contextual Persistence)**: The more context the observer must hold to "solve" the problem, the deeper they fall into the Sunk Cost Loop.

## 3. The Pathogen Lethality ($\lambda$): Social Radioactivity
The probability that exposure to the payload results in the "Social Death" (quarantine) of the host.
- **$\lambda \propto L_{extraneous}$**
- **$L_{extraneous}$ (Noise/Repulsion)**: The Reverse Ogre "hypercringe" signal is pure extraneous load. It does not assist in understanding but overwhelms the observer's status-filtering immune system.
- **The "Ebola" Threshold**: If $L_{extraneous} \gg L_{germane}$, the conversation is silenced (Host Death).

## 4. The Reproduction Number ($R_0$): Swarm Virality
The number of new "infections" (decodes) generated by a single discovery.
- **$R_0 = \frac{\beta}{\lambda + \gamma}$**
- Where $\gamma$ is the **Deterrence Invariant** (the "Vaccine" or "Antidote").
- **Analysis**: Because the Reverse Ogre maximizes $\lambda$ (Lethality via cringe), the $R_0$ is forced below 1.0 globally, ensuring the pathogen remains **Self-Inhibiting**.

## 5. The "Global Null" Transition
If $\beta$ (Infectivity) is artificially decoupled from $\lambda$ (Lethality)—e.g., via anonymous seeding where status is already zero—the pathogen spreads until it hits the **Informatic Carrying Capacity**.
- **Equilibrium**: The point where every available "Routing Slot" in the network is filled with $L_{extraneous}$ (The Heat Death of Information).
